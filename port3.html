Number Systems

What are number systems?

  • A numeral system is a writing system for expressing numbers; it is a mathematical notation for consistently representing numbers in a set using digits or other symbols (Robert, 2022).

  • Number systems are conventional systems of signs (i.e., words, digits) that have representational and analytical meanings (Lehman et al., 2013).

  • Number systems are formal systems for representing and manipulating numbers, including binary, decimal, and hexadecimal systems, which are essential in computer science (Nunes & Bryant, 2022).

Brief History of Number Sytems

Ancient Tally Marks

  • The earliest form of number representation, using simple tally marks for counting and record-keeping.

  • These marks were often carved on bones or stones.

Babylonian and Egyptian Systems:

  • The Babylonians used a base-60 (sexagesimal) number system around 2000 BCE, influencing astronomical calculations.

  • The Egyptians used a base-10 (decimal) system for administrative and construction purposes, represented with hieroglyphs.

Greek and Roman Numerals

  • The Greeks modified the Egyptian system, introducing alphabetic numerals and the concept of zero and place value.

  • The Romans used a letter-based system (I, V, X, L, C, D, M) for trade and engineering.

Hindu-Arabic Numerals

  • Developed in India around the 6th century CE, introducing zero and positional notation.

  • This system was later transmitted to the Islamic world and Europe, forming the foundation of the modern decimal system.

Non-European Contributions

  • The Mayans developed a vigesimal (base-20) system.

  • The Chinese used a rod numeral system for calculations.

Modern Digital Systems

  • With the rise of computers, binary (base-2), octal (base-8), and hexadecimal (base-16) systems became crucial for digital computing and data representation.

Components of Number Systems

Base (or Radix)

  • Determines the number of unique digits, including zero, used in the system.

  • Example:

    • Decimal system (base 10): Digits 0-9

    • Binary system (base 2): Digits 0 and 1

Digits

  • Symbols used to represent numbers in a system.

  • Each digit's value depends on its position and the base of the system.

  • Example:

    • In the decimal system, digits range from 0 to 9.

Positional Value

  • The value of a digit depends on its position in the number and the base.

  • Each position represents a power of the base.

  • Example:

    • In the number 345:

      • 5 is in the units place (base^0)

      • 4 is in the tens place (base^1)

      • 3 is in the hundreds place (base^2)

Arithmetic Operations

  • Rules for performing arithmetic operations like addition, subtraction, multiplication, and division.

  • These operations depend on the base and positional values.
